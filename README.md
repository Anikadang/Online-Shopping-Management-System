# Online Shopping Management System

## Overview
This is a console-based Online Shopping Management System developed using C++ and Object-Oriented Programming concepts.

## Features

- Administrator Login
- Customer Registration & Login
- Product Management
- Shopping Cart
- Checkout
- Payment Gateway
- Invoice Generation
- File Handling

## OOP Concepts Used

- Classes & Objects
- Inheritance
- Polymorphism
- Encapsulation
- Aggregation

## Technologies

- C++
- STL
- File Handling

## How to Run

Compile

g++ Online_Shopping_Management_System_.cpp -o shopping

Run

Windows

.\shopping

Linux

./shopping
